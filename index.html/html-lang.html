HTML Kılavuzuna Hoşgeldiniz

HTML (HyperText Markup Language), web sayfalarının yapısını oluşturan bir işaretleme dilidir. HTML, başlıklar, paragraflar, bağlantılar, resimler ve daha fazlası gibi temel öğeleri tanımlar.

HTML Öğeleri

  • <div>: Sayfa düzeni için kullanılan bir öğedir.
  • <p>: Paragraf oluşturur.
  • <a>: Bağlantı oluşturur.
  • <img>: Resim ekler.

HTML Örneği


<html>
  <head>
    <title>Web Sayfası Başlığı</title>
  </head>
  <body>
    <h1>Merhaba, Dünya!</h1>
    <p>Bu bir HTML örneğidir.</p>
  </body>
</html>
