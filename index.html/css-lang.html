CSS Kılavuzuna Hoşgeldiniz

CSS (Cascading Style Sheets), HTML öğelerini stilize etmek için kullanılır. CSS ile web sayfanızın görünümünü değiştirerek, renk, boyut, konum ve daha fazlasını kontrol edebilirsiniz.

CSS Temel Özellikleri

  • color: Yazı rengini belirler.
  • background-color: Arka plan rengini belirler.
  • margin: Öğenin dış kenar boşluğunu ayarlar.
  • padding: Öğenin iç kenar boşluğunu ayarlar.
  • flexbox: Öğeleri yatay ve dikey olarak hizalar.

CSS Örneği


body {
    background-color: #f0f0f0;
}

h1 {
    color: #5CD2B9;
}

.container {
    display: flex;
    justify-content: center;
}
